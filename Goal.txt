Zoho Creator Integration with Oracle Database via Web Service

Request Objective:
Create a Web Service that integrates an Oracle database with Zoho Creator to read item data from Oracle and automatically and periodically insert or update it within a Zoho Creator application.
The integration will be implemented with Oracle Database as the Source and Zoho as the Destination.

Technical Requirements:

Oracle:
Schema: TC2025/TC
Connect String / TNS: dreamer-laptop:1521/orcl

Zoho Creator:
Application Name: carton
Form: Items_Data

Data Mapping:

SK1MF Table in Oracle ↔ Items_Data in Zoho:

Decision Box fields of type VARCHAR2(1) in Oracle containing values (Y/N) should be converted to values (true/false) in Zoho.
